1.Οδηγίες εγκατάστασης εφαρμογής

Βήμα 1ο:  Κατεβάστε το αρχείο [app-release.apk] στο Android κινητό σας ή στον emulator.

Βήμα 2ο: Πατήστε πάνω στο αρχείο APK. Αν το κινητό σας εμφανίσει προειδοποίηση για "Άγνωστες Εφαρμογές", επιλέξτε "Να επιτρέπεται" (Allow) για αυτή την εγκατάσταση και ύστερα πατήστε το κουμπί “Εγκατασταση”.

Βήμα 3ο: Μετά την ολοκλήρωση, το εικονίδιο της εφαρμογής θα εμφανιστεί στο μενού σας.
Για να εξερευνήσετε πλήρως τις δυνατότητες της εφαρμογής, είναι αναγκαίο να ορίσετε την τοποθεσία της συσκευής σας σε ένα νησί των κυκλάδων (προτιμότερο στη  Νάξο, λόγω μεγαλύτερου όγκου αποθηκευμένων τοποθεσιών). 


Προτείνουμε να κατέβει η εφαρμογή “Fake GPS” (https://play.google.com/store/apps/details?id=com.blogspot.newapphorizons.fakegps).
Στη συνέχεια συνδεθείτε στην εφαρμογή με τον παρακάτω προ-εγκατεστημένο λογαριασμό (ή δημιουργήστε δικό σας πατώντας το Create One):
Username: GeoFlo
Password: george1

Σε αυτό το σημείο θα σας ζητηθεί να δώσετε access στο location σας.
Ανοίγοντας την εφαρμογή, βρίσκεστε στην Home Screen όπου μπορείτε να δείτε φωτογραφίες από διάφορα νησιά των κυκλάδων κάνοντας swipe, να παρακολουθήσετε την πρόοδο σας στην εφαρμογή στα Challenges και να δείτε τρέχοντα events. Πατώντας παρατεταμένα σε ένα event, αυτό γίνεται share στο Island Groupchat.
Επισκεφθείτε το Island Pass (δεύτερο κουμπί του Navigation Bar) που θα εμφανίζεται η ένδειξη “Post to View” (Island Pass Locked). Βγάζοντας μια φωτογραφία η οθόνη “ξεκλειδώνεται” και αποκτάται πρόσβαση στο Groupchat του νησιού που επιλέξατε ως τοποθεσία (σε αυτό το σημείο θα σας ζητηθεί access στην camera). Στην ξεκλειδωμένη οθόνη μπορείτε να δείτε τις φωτογραφίες που έχουν ανεβάσει οι υπόλοιποι χρήστες της εφαρμογής που βρίσκονται στο ίδιο νησί και πατώντας στο Username τους να επισκεφθείτε στο Profile τους. Στο Groupchat όπου συμμετέχουν όσοι έχουν ξεκλειδώσει το Island Pass, σας δίνεται η δυνατότητα να συνομιλίσετε με τους χρήστες και πατώντας παρατεταμένα σε ένα μήνυμα να το αντιγράψετε ή να το διαγράψετε (μόνο ως sender). Σημειώνεται ότι το Island Pass “κλειδώνει” αυτόματα κάθε 24 ώρες και ο χρήστης πρέπει να ξανααναβάσει φωτογραφία για να αποκτήσει ξανα πρόσβαση. Επίσης, σε περίπτωση που η τοποθεσία της συσκευής δεν αντιστοιχεί σε κάποιο από τα νησιά των Κυκλάδων, θα εμφανίζεται η ένδειξη “Invalid Location” και η επιλογή “Try Again”.
Στο Explore Menu στο Map της εφαρμογής (μεσαίο κουμπί του Navigation Bar) μπορείτε να εξερευνήσετε τα διάφορα τοπόσημα του νησιού που βρίσκεστε αλλα και άλλων κυκλαδονήσιων και να δείτε εικόνες και πληροφορίες για παραλίες, χώρους εστίασης και αξιοθέατα. Επίσης υπάρχει συντόμευση του Island Pass, όπου σε περίπτωση που είναι unlocked θα βλέπετε τα τελευταία μηνύματα του Groupchat.
Στη συνέχεια από το My Trips (τέταρτο κουμπί του Navigation Bar) μπορείτε να φτιάξετε ένα δυναμικό πρόγραμμα διακοπών, επιλέγοντας τοπόσημα που επιθυμείτε να επισκεφθείτε την κάθε μέρα του ταξιδιού σας. Ο διαδραστικός χάρτης εμφανίζει όλα τα επιλεγμένα σημεία ενδιαφέροντος, βοηθώντας σας να οργανώσετε τη διαδρομή σας με ευκολία. Μπορείτε ανά πάσα στιγμή να αναδιατάξετε τη σειρά επίσκεψης, να επεξεργαστείτε τις επιλογές σας ή να διαγράψετε το πρόγραμμα εξ ολοκλήρου. 
Τέλος, στο Profile Menu (πέμπτο κουμπί Navigation Bar) θα πρέπει να βλέπετε το Username σας και την φωτογραφία που ποστάρατε στο Island Pass. Από τα settings μπορείτε να αλλάξετε το Username καθώς και να ορίσετε Profile description και Profile photo. Επίσης πατώντας την φωτογραφία που έχετε ανεβάσει, υπάρχει η δυνατότητα διαγραφής της, όπου σε αυτή την περίπτωση “ξανακλειδώνει” το menu του Island Pass (σε αυτό το σημείο θα σας ζητηθεί access στην camera). Τέλος στο Profile υπάρχει counter με τον αριθμό των νησιών που έχετε επισκεφθεί και των Post που έχετε ανεβάσει.

3. Διαφοροποιήσεις από το Πρωτότυπο
Προστέθηκε η δυνατότητα επίσκεψης (σε view-only mode) σε profile άλλων χρηστών, μέσω της οθόνης του Island Pass, πατώντας πάνω στο Username τους.
Επίσης, προστέθηκε νέο feature στο Profile, με μετρητή των νησιών που έχει επισκεφθεί ο κάθε χρήστης και των post που έχει ανεβάσει.
Τέλος στο My Trips, στην ανασκόπηση ενός προγράμματος, προστέθηκε ο χάρτης για καλύτερη κατανόηση και επισκόπηση των προγραμματισμένων επισκέψεων.
